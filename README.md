# Scripts for pulsed laser deposition (PLD) optimization

This repository contains a collection of scripts for optimizing the deposition of thin films using pulsed laser deposition (PLD).

## Installation

Run `pip install -r requirements.txt` to install the required dependencies.

For Vimba, see the [Vimba Developer Guide](https://docs.alliedvision.com/Vimba_DeveloperGuide/pythonAPIManual.html#)
